
     Objects and its internal representation in JavaScript


1. The Object type represents one of JavaScript's data types.

2. Object {}

3. Loosely speaking, objects in JavaScript may be defined as an 
   unordered collection of related data,
   of primitive or reference types, in the form of “key: value” pairs. 

4. These keys can be variables or functions and are called 
   properties and methods, respectively, in the context of an object.
